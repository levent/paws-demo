PAWS (Pachube Advanced Web-scale Socket-server)

This demo shows how you can use the Pachube Advanced Web-scale Socket-server to receive realtime feed updates.

Our new api keys mean you can limit them to only allow access to public data. In other words they're safe to publish in your html source code.

Once connected to the server, we then manipulate the DOM everytime a message is received.

This is all built with HTML and Javascript.

As WebSockets are a fairly recent concept, you'll need to use a browser that is capable (Google Chrome or Safari 5).
