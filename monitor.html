PAWS (Pachube Advanced Web-scale Socket-server)

This demo uses the Pachube Advanced Web-scale Socket-server (yea that's right) to receive realtime feed updates.

Our new api keys mean you can limit them to having read-only access, making them safe to publish in your html source code.

Once connected, the server sends us messages and we use the WebSocket API's onmessage event handler to reflect the changes on the page.

This is all built using HTML5 and Javascript.

As WebSockets are a fairly recent concept, you'll need to use a browser that has support (Google Chrome or Safari 5).
